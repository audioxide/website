# Typography

Audioxide uses three fonts:

<ul>
    <li><a class="source-sans-pro" href="https://fonts.google.com/specimen/Source+Sans+Pro">Source Sans Pro</a></li>
    <li><a class="spectral" href="https://fonts.google.com/specimen/Spectral">Spectral</a></li>
    <li><a class="rounded-elegance" href="https://www.cdnfonts.com/rounded-elegance.font">Rounded Elegance</a></li>

Source Sans Pro is used for headings, Spectral for body copy, and Rounded Elegance for our logo.
